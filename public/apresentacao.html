Escola e família
mais próximas do que nunca.

Segurança & LGPD
Proteção de dados de menores e conformidade legal
Segurança contra ataques
  • Rate limiting por IP: 120 req/min padrão, 5/min em criação de usuários
  • Bloqueio de payloads acima de 1MB (DoS prevention)
  • Headers HTTP: X-Frame-Options, HSTS, X-Content-Type-Options
  • Validação de UUIDs e tamanho máximo de todos os campos
  • Autenticação Supabase Auth com JWT em todas as rotas
LGPD — Proteção de menores
  • Dados dos alunos acessíveis apenas pelo vínculo cadastrado (responsável específico)
  • Row Level Security (RLS) no banco de dados PostgreSQL
  • APIs não retornam dados de terceiros — cada usuário vê apenas o seu
  • Notificações push com dados mínimos necessários
  • Cache-Control: no-store em todas as APIs — dados não ficam em cache
Dados de menores: somente o responsável vinculado formalmente ao aluno pode ver as informações de presença, notas e histórico.
