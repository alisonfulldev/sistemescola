Segurança & Qualidade
Construído para durar

Cada detalhe foi pensado para garantir que os dados da escola estejam sempre protegidos e acessíveis.

🔐

Row Level Security

RLS no Supabase garante que cada usuário só acessa seus próprios dados, mesmo em requisições diretas.

📋

Logs de Auditoria

Todas as ações importantes são registradas com usuário, horário e resultado para rastreabilidade total.

⏱️

Sessão com Timeout

Sessões expiram automaticamente após 10 minutos de inatividade com aviso de 1 minuto antes.

✅

Validação Zod

Todos os dados enviados para a API são validados com schemas Zod antes de qualquer operação.

🔄

Tempo Real

Atualizações via Supabase Realtime. O dashboard atualiza automaticamente sem precisar recarregar.

📱

PWA — Mobile First

Funciona como app instalável no celular. Push notifications chegam mesmo com o app fechado.

🏫

Multi-escola

Dados isolados por escola. Cada diretor e secretaria acessa apenas os dados da sua unidade.

🗄️

Backup PostgreSQL

Scripts de backup completo com pg_dump. Mantém os últimos 10 backups com agendamento automático.
