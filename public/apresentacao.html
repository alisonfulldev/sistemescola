O problema

Como a escola funciona
sem um sistema

Processos manuais geram erros, atrasos e falta de visibilidade. Conheça os problemas que o EstudApp elimina.

📝

Chamada no papel

Lenta, propensa a erros e sem histórico digital. O professor perde tempo que deveria usar para ensinar.

👁️

Sem visibilidade em tempo real

Diretor e secretaria só descobrem as faltas no dia seguinte, quando já é tarde para agir.

📵

Responsável no escuro

O pai só fica sabendo da falta do filho dias depois — ou nunca. Sem canal de comunicação direto.

A solução

EstudApp resolve tudo isso
em 3 passos simples

Do cadastro ao alerta automático, o fluxo acontece sozinho. Você só gerencia as exceções.

Passo 01
📋

Professor abre o celular

Inicia a chamada pelo app, marca presença ou falta de cada aluno em segundos. Concluído — histórico salvo automaticamente.

Passo 02
🔔

Responsável é notificado

Assim que a falta é registrada, o responsável recebe uma notificação push no celular — mesmo com o app fechado.

Passo 03
📊

Gestão acompanha ao vivo

Diretor e secretaria veem chamadas, frequência e alertas em tempo real, sem precisar perguntar para ninguém.

Interface

O dashboard que a gestão precisa

Chamadas do dia, alertas e frequência — tudo em uma só tela, atualizado em tempo real.

Segurança & Qualidade

Construído para durar

Cada detalhe foi pensado para que os dados da escola estejam sempre protegidos e acessíveis.

🔐

Row Level Security

RLS no Supabase garante que cada usuário só acessa seus próprios dados, mesmo em requisições diretas ao banco.

📋

Logs de Auditoria

Todas as ações são registradas com usuário, horário e resultado para rastreabilidade total.

⏱️

Timeout de Sessão

Sessões expiram após 10 minutos de inatividade, com aviso de 1 minuto antes.

✅

Validação Zod

Todos os dados enviados à API são validados com schemas Zod antes de qualquer operação.

🔄

Tempo Real

Dashboard atualiza automaticamente via Supabase Realtime sem precisar recarregar a página.

📱

PWA Mobile First

Instalável no celular. Push notifications chegam mesmo com o app fechado.

🏫

Multi-escola

Dados isolados por escola. Cada diretor e secretaria acessa apenas sua unidade.

🗄️

Backup PostgreSQL

Scripts automáticos com pg_dump. Mantém os últimos 10 backups com agendamento diário.
